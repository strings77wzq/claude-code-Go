# Feedback

Have feedback? We'd love to hear from you.

We use GitHub to manage all feedback. Please use the appropriate channel below for your type of contribution.

---

## Bug Reports

Found a bug? Help us fix it!

**Channel:** [GitHub Issues](https://github.com/strings77wzq/claude-code-Go/issues)

Use GitHub Issues to report:
- Crashes or panics
- Unexpected behavior or errors
- Incorrect functionality
- Performance issues

**Before submitting:**
- Search existing issues to avoid duplicates
- Include reproduction steps if possible
- Mention your Go version and OS

---

## Feature Requests

Have an idea for a new feature? We'd love to hear it!

**Channel:** [GitHub Discussions](https://github.com/strings77wzq/claude-code-Go/discussions)

Use GitHub Discussions to suggest:
- New features or capabilities
- Improvements to existing functionality
- Integration ideas
- UI/UX enhancements

**Before submitting:**
- Search existing discussions to avoid duplicates
- Describe the problem your feature would solve
- Consider how it fits with the project vision

---

## Other Ways to Contribute

- ⭐ **Star the project** on GitHub
- 📖 **Improve documentation** via Pull Requests
- 🐛 **Fix bugs** by submitting Pull Requests
- 💬 **Join discussions** and help answer questions
