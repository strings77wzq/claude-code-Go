# Pricing

claude-code-Go is open source and free to use. There are no paid plans, enterprise SKUs, or priority-support commitments in the verified product scope.

## Open Source (Free)

Current verified and partial open-source scope:

- Full agent loop with 10 built-in tools
- Multi-provider support (Anthropic, OpenAI, and more)
- Permission system with 3-tier model
- Session persistence and resume
- SSE streaming
- MCP integration (Partial v0.3)
- LSP support (Partial v0.3)
- Auto-recovery
- Auto-update

**Cost**: Free (MIT License)

## Parked Business Concepts

The following ideas are parked until core v0.3 extension productization has stronger evidence:

- Pro plan

No pricing, timeline, or support promise is currently attached to those concepts.
