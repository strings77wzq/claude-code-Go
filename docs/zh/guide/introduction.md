# 概述

欢迎了解 claude-code-Go，这是一个用 Go 实现的 Anthropic Claude Code（Cline）智能体系统。

## 什么是 claude-code-Go？

claude-code-Go 是一个终端驱动的 AI 编程助手，它将 Claude Code 强大的智能体能力带入 Go 生态系统。它完整实现了智能体循环，包含内置工具、权限系统、MCP（Model Context Protocol）支持以及 SSE（Server-Sent Events）流式输出。

你可以把它想象成一个 AI 编程伙伴，能够：
- 读取、编写和编辑项目中的文件
- 执行 shell 命令来运行构建、测试或脚本
- 使用 glob 模式和内容搜索来浏览代码库
- 在终端 REPL 中与你交互式地协作

## 它和普通聊天机器人有什么不同？

与传统的聊天机器人不同，claude-code-Go 作为一个具有自主能力和工具使用能力的**智能体**运作。

| 特性 | 聊天机器人 | claude-code-Go |
|------|------------|----------------|
| **交互模式** | 请求-响应 | 持续智能体循环，调用工具 |
| **能力** | 仅文本生成 | 读取、编写、编辑、执行、搜索文件 |
| **状态管理** | 无状态 | 消息历史 + 上下文压缩 |
| **用户控制** | 被动接收 | 危险操作需主动审批 |
| **使用场景** | 问答、对话 | 代码开发、自动化 |

## 技术栈

claude-code-Go 遵循最小依赖的原则构建：

| 组件 | 技术 |
|------|------|
| **编程语言** | Go 1.23+ |
| **标准库** | 仅使用 Go 标准库（主二进制无外部依赖） |
| **Python 测试框架** | 可选（用于测试、模拟、评估） |
| **API** | Anthropic Messages API + SSE 流式传输 |

`harness/` 目录下的 Python 测试框架是可选的，提供：
- 模拟 API 服务器（测试时无需 API 费用）
- 会话回放（用于调试）
- 质量评估器（用于集成测试）

## 核心数字

- **约 31** 个 Go 源文件（非测试）
- **约 4,000** 行 Go 代码
- **约 2,000** 行 Python 代码（harness）
- **8** 个内部模块
- **6** 个内置工具（Read、Write、Edit、Glob、Grep、Bash）

## 你将学到什么

本文档涵盖 claude-code-Go 的架构和实现细节：

1. **AI 智能体架构** — 智能体循环如何工作、决策过程和工具选择

3. **工具系统** — 内置工具 + MCP 集成，扩展能力
4. **安全设计** — 三层权限系统，控制危险操作
5. **工程实践** — 结构化日志、优雅关闭、配置管理
