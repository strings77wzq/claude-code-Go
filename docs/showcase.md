# Showcase

Example use cases for claude-code-Go.

## Use Cases

### Code Review

Use claude-code-Go to review your code before submitting PRs:

```bash
go-code -p "Review this code for Go best practices and potential bugs" < main.go
```

### Documentation

Generate documentation for your codebase:

```bash
go-code -p "Generate README documentation for this project"
```

### Refactoring

Refactor large codebases with confidence:

```bash
go-code -p "Refactor this to use interfaces instead of concrete types"
```

### Learning

Learn new codebases quickly:

```bash
go-code -p "Explain the architecture of this codebase"
```

### Testing

Generate test cases:

```bash
go-code -p "Write unit tests for this function"
```
