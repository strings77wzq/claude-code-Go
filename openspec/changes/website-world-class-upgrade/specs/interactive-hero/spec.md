## ADDED Requirements

### Requirement: Interactive hero section
The homepage hero SHALL feature a terminal typewriter animation simulating real REPL interaction.

#### Scenario: Typewriter animation
- **WHEN** a user visits the homepage
- **THEN** the hero terminal shows a simulated REPL session with typewriter effect

#### Scenario: Cursor blinking
- **WHEN** the typewriter animation completes
- **THEN** a blinking cursor is shown at the prompt
