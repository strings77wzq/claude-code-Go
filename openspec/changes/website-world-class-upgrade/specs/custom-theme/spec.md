## ADDED Requirements

### Requirement: Custom VitePress theme
The site SHALL use a deeply customized VitePress theme with brand colors and animations.

#### Scenario: Brand colors
- **WHEN** a user views the site
- **THEN** Go blue (#00ADD8) is used as the primary brand color throughout

#### Scenario: Scroll animations
- **WHEN** a user scrolls to a section
- **THEN** content fades in with a slide-up animation
