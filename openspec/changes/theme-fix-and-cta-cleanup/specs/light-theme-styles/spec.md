## MODIFIED Requirements

### Requirement: Light theme styles
All custom components SHALL display correctly in both dark and light themes.

#### Scenario: Terminal window in light theme
- **WHEN** a user views the terminal window in light theme
- **THEN** it remains dark (terminal convention) with proper contrast

#### Scenario: Feature blocks in light theme
- **WHEN** a user views feature blocks in light theme
- **THEN** they have proper background, border, and text colors
