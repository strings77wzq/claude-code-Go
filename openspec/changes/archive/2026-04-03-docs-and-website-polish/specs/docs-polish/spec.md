## ADDED Requirements

### Requirement: Professional documentation language
All documentation SHALL use precise engineering language.

#### Scenario: Architecture overview
- **WHEN** a user reads the architecture overview
- **THEN** it uses precise engineering terms: "stop_reason state machine", "tool call chain", "permission gate"

#### Scenario: Design philosophy
- **WHEN** a user reads the design philosophy
- **THEN** it uses analogies and comparisons to deepen understanding
