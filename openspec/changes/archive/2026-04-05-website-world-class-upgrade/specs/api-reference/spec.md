## ADDED Requirements

### Requirement: API reference documentation
The site SHALL have a complete API reference for all tools, commands, and configurations.

#### Scenario: Tool reference
- **WHEN** a user visits the API reference
- **THEN** they see all 10 built-in tools with parameters, examples, and permission levels

#### Scenario: Command reference
- **WHEN** a user visits the command reference
- **THEN** they see all REPL commands with descriptions and examples
