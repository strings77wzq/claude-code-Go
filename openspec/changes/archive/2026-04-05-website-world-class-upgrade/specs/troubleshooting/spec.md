## ADDED Requirements

### Requirement: Troubleshooting guide
The site SHALL have a comprehensive troubleshooting guide.

#### Scenario: FAQ section
- **WHEN** a user visits the troubleshooting page
- **THEN** they see common issues with solutions (API key errors, network issues, permission denied)

#### Scenario: Error code reference
- **WHEN** a user encounters an error
- **THEN** they can look up the error code and find the solution
