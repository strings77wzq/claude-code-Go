## ADDED Requirements

### Requirement: User onboarding
The homepage SHALL guide new users through a quick start flow.

#### Scenario: Role selection
- **WHEN** a user views the homepage
- **THEN** they see "Choose your role" cards (Full-stack, Architect, Student)

#### Scenario: Quick start steps
- **WHEN** a user clicks a role card
- **THEN** they see a tailored 3-step quick start guide
