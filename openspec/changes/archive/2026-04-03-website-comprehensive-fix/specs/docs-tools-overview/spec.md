## ADDED Requirements

### Requirement: Tools overview page
The website SHALL have a tools system overview page.

#### Scenario: Tool interface design
- **WHEN** a user views the tools overview
- **THEN** they see: Tool interface definition with 5 methods explained, Registry pattern, extension guide

#### Scenario: Built-in tools table
- **WHEN** a user views the tools overview
- **THEN** they see a table of all 6 built-in tools with purpose, permission requirement, and file location
