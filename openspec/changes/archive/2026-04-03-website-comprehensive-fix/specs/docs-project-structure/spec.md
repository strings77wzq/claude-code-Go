## ADDED Requirements

### Requirement: Project structure page
The website SHALL have a detailed project structure page.

#### Scenario: Directory tree
- **WHEN** a user views the project structure page
- **THEN** they see a complete directory tree with emoji icons for each module

#### Scenario: Module breakdown
- **WHEN** a user reads the module section
- **THEN** they see: module name, core classes, one-line description for each

#### Scenario: Dependency relationships
- **WHEN** a user reads the dependency section
- **THEN** they see a clear dependency diagram showing unidirectional dependencies
