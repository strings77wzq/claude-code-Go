## ADDED Requirements

### Requirement: Core code documentation
The website SHALL have core code documentation pages explaining key implementations.

#### Scenario: Entry point page
- **WHEN** a user views the entry point page
- **THEN** they see: main.go walkthrough, component initialization sequence, signal handling explanation

#### Scenario: Agent Loop implementation page
- **WHEN** a user views the Agent Loop implementation page
- **THEN** they see: loop code walkthrough, stop_reason handling, tool execution flow, history management
