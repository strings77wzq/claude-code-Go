## ADDED Requirements

### Requirement: README includes visual demo
The README SHALL include a Demo GIF section showing actual usage.

#### Scenario: Demo section exists
- **WHEN** a visitor opens the README
- **THEN** they see a "Demo" section with animated GIF
- **AND** the GIF shows real usage of the tool

#### Scenario: Installation instructions clear
- **WHEN** a visitor reads the README
- **THEN** they can install and run the tool in < 5 minutes

#### Scenario: Quick start guide
- **WHEN** a new user wants to try the tool
- **THEN** they find a clear quick start section
