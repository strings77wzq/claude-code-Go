## MODIFIED Requirements

### Requirement: Theme switching
The site SHALL support dark/light theme switching with system default option.

#### Scenario: Theme toggle
- **WHEN** a user clicks the theme toggle
- **THEN** they can switch between dark, light, and system preference

#### Scenario: System preference
- **WHEN** a user selects "auto" or "system"
- **THEN** the theme matches their OS preference
