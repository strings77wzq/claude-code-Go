## MODIFIED Requirements

### Requirement: Theme switching works correctly
The site SHALL support dark/light/system theme switching.

#### Scenario: System default
- **WHEN** a user first visits the site
- **THEN** the theme matches their OS preference

#### Scenario: Manual toggle
- **WHEN** a user clicks the theme toggle
- **THEN** they can switch between dark and light themes

#### Scenario: Preference persistence
- **WHEN** a user returns to the site
- **THEN** their theme preference is remembered
