## ADDED Requirements

### Requirement: Issue templates
The repository SHALL have issue templates for bug reports and feature requests.

#### Scenario: Bug report template
- **WHEN** a user opens a new issue
- **THEN** they see a Bug Report template with environment, steps to reproduce, expected/actual behavior

#### Scenario: Feature request template
- **WHEN** a user opens a new issue
- **THEN** they see a Feature Request template with problem description, proposed solution
