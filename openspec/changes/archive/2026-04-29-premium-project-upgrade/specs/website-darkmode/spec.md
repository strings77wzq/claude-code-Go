## ADDED Requirements

### Requirement: Dark mode is optimized
The website SHALL have a polished dark mode.

#### Scenario: Smooth transitions
- **WHEN** a user toggles theme
- **THEN** they see smooth color transitions

#### Scenario: System preference detection
- **WHEN** a user first visits
- **THEN** theme matches system preference

#### Scenario: Explicit toggle
- **WHEN** a user wants to switch
- **THEN** they find an obvious toggle button
