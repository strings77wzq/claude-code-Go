## ADDED Requirements

### Requirement: Interactive playground exists
The website SHALL have an interactive code playground.

#### Scenario: Embedded editor
- **WHEN** a user visits the playground
- **THEN** they see an embedded code editor

#### Scenario: Copy functionality
- **WHEN** a user views code
- **THEN** they can copy with one click

#### Scenario: Live output
- **WHEN** a user runs examples
- **THEN** they see output in real-time
