## ADDED Requirements

### Requirement: Real demo recordings exist
The project SHALL have actual demo recordings, not placeholders.

#### Scenario: Asciinema recordings created
- **WHEN** a user visits the README or docs
- **THEN** they see actual asciinema recordings of the tool in action

#### Scenario: Multiple demo scenarios
- **WHEN** a user wants to see different use cases
- **THEN** they find at least 5 different demo recordings

#### Scenario: Demo GIF generated
- **WHEN** a user views the README
- **THEN** they see an animated GIF showing real usage
