## ADDED Requirements

### Requirement: Skills tutorial
The website SHALL have a technical tutorial for the Skills system.

#### Scenario: Skills tutorial content
- **WHEN** a user visits the Skills tutorial page
- **THEN** they see: what Skills are, how to create them (JSON format), how to register them, examples

#### Scenario: Example Skills
- **WHEN** a user reads the tutorial
- **THEN** they see complete examples: review-pr, explain-code, write-tests
