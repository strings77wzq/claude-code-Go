## ADDED Requirements

### Requirement: Project introduction page
The website SHALL have a project introduction page as the first page of the guide.

#### Scenario: English introduction
- **WHEN** a user visits the English guide section
- **THEN** they see: what the project is, how it differs from chatbots, tech stack table, core numbers, learning outcomes, target audience

#### Scenario: Chinese introduction
- **WHEN** a user visits the Chinese guide section
- **THEN** they see the same content in professional Chinese

#### Scenario: Core numbers display
- **WHEN** a user views the introduction page
- **THEN** they see: file count, module count, tool count, LOC count
