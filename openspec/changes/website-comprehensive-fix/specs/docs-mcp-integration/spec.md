## ADDED Requirements

### Requirement: MCP integration documentation
The website SHALL have a dedicated MCP integration page.

#### Scenario: MCP protocol explanation
- **WHEN** a user views the MCP page
- **THEN** they see: what MCP is, stdio transport, JSON-RPC format, tool discovery process

#### Scenario: Implementation details
- **WHEN** a user reads the MCP implementation section
- **THEN** they see: transport layer, client implementation, tool adapter, configuration
