claude-code-Go

AI-powered coding assistant with a full agent loop, tool execution, and permission management — built in Go.

Get Started
View on GitHub
🔄

Agent Loop

Autonomous "think → act → observe" cycle driven by stop_reason dispatch. Handles tool_use, end_turn, and max_tokens seamlessly.

🛠️

6 Built-in Tools

Read, Write, Edit, Glob, Grep, and Bash — a complete toolset for software engineering tasks out of the box.

🔒

Permission System

Three-tier permission model (ReadOnly / WorkspaceWrite / DangerFullAccess) with rule-based matching and session memory.

🔌

MCP Integration

Model Context Protocol support with stdio transport, JSON-RPC client, and automatic tool discovery from external servers.

🌊

SSE Streaming

Real-time token-by-token response streaming with custom SSE parser. No external dependencies needed.

🧠

Context Management

Intelligent token estimation and automatic conversation compaction. Preserves context while staying within model limits.
