为什么选择 Go？ ​

GoPythonRust
部署方式单二进制文件需要运行时环境单二进制文件
外部依赖零依赖pip installCargo build
交叉编译GOOS=linux go build平台相关需要交叉工具链
并发模型Goroutine（内置）asyncio（库）async/await
学习曲线适中简单陡峭

Go 兼具两者的优势：像 Rust 一样的单二进制部署，接近 Python 的开发效率。
