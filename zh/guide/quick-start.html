快速开始 ​

本指南将带您完成首次运行 go-code 的完整流程。

前提条件 ​

在运行 go-code 之前，您需要：

  1. 一个 Anthropic API 密钥
  2. 已编译完成的二进制文件（参见安装指南）

配置 API 密钥 ​

方式一：环境变量 ​

bash
export ANTHROPIC_API_KEY=sk-ant-your-api-key-here
1

建议将此配置添加到您的 shell 配置文件（~/.bashrc、~/.zshrc 等）中以实现持久化。

方式二：配置文件 ​

创建 ~/.go-code/settings.json：

json
{
  "apiKey": "sk-ant-your-api-key-here"
}
1
2
3

配置加载器按以下顺序搜索（后者覆盖前者）：

  1. CLI 参数（最高优先级）
  2. 环境变量：ANTHROPIC_API_KEY
  3. 项目配置：.go-code/settings.json
  4. 用户配置：~/.go-code/settings.json
  5. 内置默认值（最低优先级）

运行 go-code ​

启动交互式会话：

bash
./bin/go-code

单次命令模式 ​

用于执行一次性任务：

bash
./bin/go-code "创建一个 Go 的 hello world 程序"
1

此命令将执行完成后退出。

可用命令 ​

在交互模式下，您可以使用以下特殊命令：

命令说明
/help显示可用命令
/clear清除对话历史
/exit退出程序
/quit退出程序（与 /exit 相同）
/model显示当前模型

启动参数 ​

go-code 支持以下启动选项：

位置参数 ​

bash
go-code [prompt]
1
  • prompt（可选）：如果提供，go-code 将执行此单次命令然后退出

环境变量 ​

变量说明必需
ANTHROPIC_API_KEY您的 Anthropic API 密钥是
ANTHROPIC_BASE_URL覆盖默认 API 端点（可选）否
ANTHROPIC_MODEL指定模型（默认：claude-3-5-sonnet-20241022）否

配置文件 ​

创建 ~/.go-code/settings.json 用于持久化配置：

json
{
  "apiKey": "sk-ant-your-api-key-here",
  "model": "claude-sonnet-4-20250514",
  "baseUrl": "https://api.anthropic.com"
}

权限提示 ​

当工具需要执行潜在危险操作时，go-code 会提示您确认权限：

go-code> 删除当前目录下的所有文件

⚠️ 此操作将删除 3 个文件。是否批准？(yes/no)：no
1
2
3

输入 yes 表示批准，输入 no 表示拒绝。

权限系统控制：

  • 文件删除和覆盖
  • Shell 命令执行
  • 网络请求
  • 其他潜在危险操作
