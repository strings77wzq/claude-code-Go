Methodology ​

All benchmarks were run on:

  • OS: Ubuntu 22.04 LTS
  • CPU: AMD Ryzen 9 5900X
  • RAM: 32 GB DDR4
  • Network: 100 Mbps

Each test was run 5 times and averaged. Cold start tests were run after system reboot.
